# Blinkit-Sales-Dashboard
This project is an interactive Blinkit Sales Dashboard built using Power BI. The dashboard provides a comprehensive view of sales performance, customer ratings, outlet distribution, and product insights, helping stakeholders make data-driven decisions.
🎯 Objectives
Analyze overall sales performance.
Compare sales across outlet locations and outlet sizes.
Identify top-performing item categories.
Track outlet establishment trends over the years.
Provide interactive filtering for better business analysis.
